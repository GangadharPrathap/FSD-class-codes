Ways to define the Colors in CSS
  • nmed colors
    • there are 147 named colors in the CSS library
  • RGB/RGBA (red,green,blue)
    • the rgb is used in order to customise the color we need to apply to our website. The representation will be in the format rgb(0,0,0). the range of each value will be 0-255. in the above other representation we have mentioned 'a' which states the transparency of the color it is called as Alpha.
  • HEX/HEX
    • HEX stands for the hexadecimal which means it has 16 values. The representation of this hex will be #000000 where each digit refer to 16 bit values
    we can also use the shortend property which has only three values than the general code value like in above #000
  • HSL/HSLA
    • In HSL the h stands for hue
    s stands for saturation
    l stands for light/lightness
