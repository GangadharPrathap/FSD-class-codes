the relative position tag in the css ia used in order to modify the position of the respective div block according to the four parameters
'top' 'bottom' 'left' 'right' even though we give the negative values.

  • top
    • Here if top(+ve) it moves down for given value
    if top(-ve) it moves up for the given value
  • Bottom
    • Here if Bottom(+ve) it moves upward for given value
    if Bottom(-ve) it moves downward for the given value
  • Left
    • Here left(+ve) it moves right to given value
    if left(-ve) it moves left to the given value
  • Right
    • Here Right(+ve) it moves left to the given value
    if right(-ve) it moves to right to the given value

the absolute position is used to modify the position of the block inside a predefined relatively positioned color block from the corners

this tag is used in the css in order to prevent the movement of the block according to scrolling in the website

The sticky Position is a combination of relative and fixed positions
until the respective boundaries of the parent the block of the sticky position it acts like a relative Postion but once the div block crosses the boundary of the parent tag it acts like a fixed Postion

the static is a position which has a default value for its position and cannot be having triggers like top left bottom right
it cannot be moved directly
