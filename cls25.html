Today's topic: Flex

the default flex direction will be into row. while using flex concept we come across another three concepts called justify content and align items and Flex-wrap.

  1. Justify-content: it is used to modify the content which is present the row wise arrangement.
  2. Align-items: it is used to modify the content which is present in the column wise arrangement.
  3. Flex-wrap: it is used to wrap the div tags or block in the interface.
  4. Flex-direction: it used to choose the direction of the orientation of blocks or div tags in the website
  5. Gap: it is used to provide a customised amount of gap between them.
