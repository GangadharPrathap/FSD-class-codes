Today's topic:
Grid

the grid is used to divide our respective block into a matrix with m rows and n columns
Properties:

  1. Always the block filled in the particular part of the grid
  2. the columns and rows in the bloack are divided on basis of frame
  3. the children are always place at the top-left part of the position
  4. the position of the children can be modified using justify-self and align-self
  5. the frame declaration can be declared linearly or can be used a function called repeat() in order ot declare n number of frames
